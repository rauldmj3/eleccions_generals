5 Consultes Simples
1-Selecciona l'identificatiu de la candidatura, el nom curt i el nom llarg de les candidatures 
les qual en el seu nom llarg és trobi la seva autodenominació com a partit.
Ordena per el nom curt i el nom llarg respectivament.

SELECT candidatura_id, nom_curt, nom_llarg
FROM candidatures
WHERE nom_llarg LIKE '%PARTI%'
ORDER BY nom_curt,nom_llarg;

2-Extreu el codi de l'institut Nacional d'Estadística i el nom de les comunitats autònomes que estan composades per més d'una paraula.
Ordena per el nom de forma descendent.

SELECT codi_ine, nom
FROM comunitats_autonomes
WHERE CHAR_LENGTH(nom) - LENGTH(REPLACE(nom, ' ', '')) > 0
ORDER BY nom DESC;

3-Selecciona els nom i cognom,sexe,data de naixement de les persones nascudes 
posteriorment al 4 de gener de 1977(data en la que es va derogar el sistema polític franquista 
i es van convocar les primeres eleccions democràtiques). Ordena per el sexe i cognom respectivament.
SELECT nom,cog1,sexe,data_naixement
FROM persones
WHERE data naixement>'04-01-1977'
ORDER BY sexe,cog1;

4-Selecciona el número de candidats com a 'num_candidats' que hi ha per cada provincia i ordena per l'identificatiu de la candidatura.
(num_candidatures potser)
SELECT candidatura_id, COUNT(*) AS num_candidats, provincia_id
FROM candidats
GROUP BY candidatura_id, provincia_id
ORDER BY candidatura_id;

5-Selecciona totes les columnes dels municipis en els que hi hagin 3 vocals o més en el seu nom.
Ordena per l'identificatiu de la provincia i consequentment per el nom.

SELECT *
FROM municipis
WHERE (nom LIKE '%a%' OR nom LIKE '%e%' OR nom LIKE '%i%' OR nom LIKE '%o%' OR nom LIKE '%u%'OR nom LIKE '%A%'OR nom LIKE '%E%'OR nom LIKE '%I%'OR nom LIKE '%O%'OR nom LIKE '%U%')
GROUP BY municipi_id
HAVING COUNT(*) >= 3
ORDER BY provincia_id, nom;

6-Selecciona l'identificaió de candidat, l'identificaió de candidat, l'identificaió de provincia  i defineix un nou valor aleatori anomenat 'Salari_anual' comprès entre els valors 14000 i 88000. 
Ordena per salari de forma descendent.

SELECT candidat_id, candidatura_id, persona_id, provincia_id, num_ordre, tipus,
FLOOR(RAND() * (88000 - 14000 + 1) + 14000) AS Salari_anual
FROM candidats
ORDER BY Salari_anual DESC;
